Webserv

Webserv is a custom HTTP server written entirely in C++ 98 as part of the 42 curriculum.

The objective of this project is to build a fully functional, non-blocking web server from scratch, without using any external web server libraries. The server must handle multiple clients simultaneously using a single I/O multiplexing mechanism.

./webserv [configuration_file]

Core Features

  • Parse NGINX-inspired config
  • Handle multiple ports/blocks
  • Serve static websites
  • Support file uploads
  • GET, POST, and DELETE methods
  • Execute CGI (PHP/Python)

Technical Excellence

  • Non-blocking I/O event loop
  • Multiplexing (select, poll, etc.)
  • Chunked transfer decoding
  • Memory safety (C++ 98)
  • Resilient under stress

Configuration Capabilities

The configuration file allows for granular control over the server's behavior, including defining interface/port pairs, limiting client body size, and setting up complex routing with directory listing and CGI support.
